Sample scripts for the FLOW/INTERFACE Winter School 2019.
There is a python script or a jupyter notebook available for both cases.
Use whichever is more convenient.

1. PINN_Burgers: contains scripts or jupyter notebook to solve the Burgers equation
2. PINN_KS: contains scripts or jupyter notebook to solve the KS equation
3. PIESN_Lorenz: contains scripts or jupyter notebook to model the Lorenz system
4. PIESN_MFE: contains scripts or jupyter notebook to model the MFE model
